# 📂 getmoney フォルダ構成ガイド

---

## 🔒 絶対に消してはいけないもの（システムの中枢）

| ファイル / フォルダ | 役割 |
|---|---|
| `generate_post.js` | スライド自動生成エンジン（核心） |
| `night_batch.js` | 夜間バッチ生成スクリプト |
| `share_slides.js` | スマホ転送サーバー |
| `batch_areas.json` | 全エリアのスポットデータベース |
| `slide_generator.html` | スライドテンプレート（標準） |
| `slide_generator_romantic.html` | スライドテンプレート（ロマンチックゴールド） |
| `slide_generator_highres.html` | スライドテンプレート（高解像度） |
| `🌙夜間バッチ生成.bat` | 夜間バッチ起動ボタン |
| `🧹投稿後クリーンアップ.bat` | 安全に生成物を全削除するボタン |
| `index.html` / `app.js` | ダッシュボード |
| `gas_scripts/` | GAS連携スクリプト |
| `.agent/config.json` | GAS URL等の設定 |
| `package.json` / `node_modules/` | Node.js依存パッケージ |
| `GEMINI.md` | プロジェクトのルール |
| `フォルダ説明.md` | このファイル |

---

## 🗑️ 投稿後に消してOKなもの

| ファイル / フォルダ | 役割 | 削除タイミング |
|---|---|---|
| `content_library/[エリア名]/` | 収集した画像・生成したスライドPNG・caption.txt | **スマホに保存 & スプレッドシートに同期済み** なら削除可 |
| `ready_to_post_zips/` | 完成ZIPファイル | スマホに転送済みなら削除可 |
| `docs/` | GitHub Pages用の一時コピー | GitHub にpush済みなら削除可 |
| `test_*.js` / `test_images.json` | 開発時のテスト残骸 | いつでも削除可 |
| `error.log` | エラーログ | いつでも削除可 |

---

## 🧹 クリーンアップ方法

投稿が終わったら **`🧹投稿後クリーンアップ.bat`** をダブルクリックするだけ！

自動で以下を削除します：
- `content_library/` の各エリアフォルダ（画像・スライド）
- `ready_to_post_zips/` の全ZIPファイル
- `docs/` の一時コピー
- テスト残骸ファイル

スクリプト・設定・テンプレートは**絶対に消しません**。

---

## 💡 運用フロー

```
🌙 夜間バッチ生成.bat を実行
        ↓
    生成完了（content_library/ に素材蓄積）
        ↓
node share_slides.js [フォルダ] でスマホに転送
        ↓
    TikTokに投稿 ✅
        ↓
🧹 投稿後クリーンアップ.bat で掃除
        ↓
    次のバッチへ！
```
